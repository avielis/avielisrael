Outils utilisés

Microsoft 365

Microsoft 365 est une suite de services cloud qui inclut les applications Office (Word, Excel, Outlook, etc.), des outils collaboratifs (Teams, OneDrive, SharePoint) et des solutions de sécurité et de gestion IT.

Sophos Connect

Sophos Connect est notre client VPN d’accès à distance gratuit conçu spécifiquement pour permettre aux télétravailleurs d’accéder à votre réseau, protégé par XG Firewall

Active Directory

Active Directory (AD) est un service d’annuaire sur Windows Server permettant la gestion des utilisateurs, groupes et ressources réseau. Il centralise les permissions et le contrôle d’accès en stockant les données sous forme d’objets.

3cx

3CX est une solution professionnelle de téléphonie IP qui permet de passer et recevoir des appels à partir d'un téléphone IP, d'un softphone, d'un smartphone (à partir d'une application mobile), ou directement depuis un navigateur web

GLPI

GLPI est un logiciel libre de gestion ITSM et de parc informatique, permettant de gérer les demandes, les équipements et la maintenance.

EXEL

Microsoft Excel est un tableur qui fait partie de la suite bureautique Microsoft Office. Cet outil permet de créer des feuilles de calcul où il est possible de réaliser des calculs complexes, des analyses de données et des graphiques

DATTO

Datto RMM est une plateforme cloud sécurisée pour surveiller, gérer et automatiser la maintenance des endpoints à distance, optimisant l’efficacité IT et réduisant les coûts.

Sharepoint

Sharepoint est une plateforme collaborative développée par Microsoft. Basée sur un site web, elle permet aux équipes professionnelles de gérer et d’échanger des contenus divers (documents, applications, etc.)..

Exchange

Microsoft Exchange est une solution de messagerie et de collaboration développée par Microsoft. Elle est principalement utilisée dans les entreprises pour gérer les e-mails, calendriers, contacts et tâches.

Putty

PuTTY est un client SSH et telnet pour Windows permettant de se connecter à des serveurs distants de manière sécurisée pour l'administration et l'exécution de commandes.
